# 病程管家｜UI/UX 设计规范文档（V1）

# 一、产品 UI 核心定位

## 产品定位

病程管家不是：

- 医院后台系统
- 医疗管理系统
- 医疗 HIS 系统
- 冷冰冰的数据工具

而是：

# 「长期病程陪伴工具」

---

## UI 核心方向

整体视觉方向：

# 轻医疗 + 温和陪伴感

关键词：

- 温和
- 干净
- 克制
- 有呼吸感
- 长期陪伴
- 不压抑
- 医疗感适中

---

# 二、当前 UI 存在的问题

---

## 1. 灰色过多

当前问题：
- 背景灰
- 卡片灰
- 分割线灰

导致：
- 页面压抑
- 没重点
- 像后台系统

---

## 2. 信息层级不明显

当前：
- 当前状态
- 提醒
- 摘要

视觉权重接近。

导致：
- 用户不知道先看哪里

---

## 3. 时间轴信息密度不合理

当前：
- 留白过大
- 时间线存在感弱
- 卡片像孤岛

---

## 4. 趋势分析页面过空

当前：
- 图表区域太空
- 缺少趋势结论
- 缺少视觉重点

---

## 5. 病程摘要缺少“整理感”

当前：
像数据库记录。

目标：
应该像：

# 「系统自动帮用户整理好的病情摘要」

---

# 三、整体 UI 设计原则

---

# 1️⃣ 首页 = 病程控制中心

首页不是个人中心。

用户打开后：

# 3秒内知道：

- 当前病情
- 最近变化
- 下次治疗时间
- 是否需要复查

---

# 2️⃣ 时间轴 = 产品核心

所有病程内容统一进入时间轴。

包括：
- PSA
- 检查
- 打针
- 用药
- 复诊
- 备注

---

# 3️⃣ 所有记录必须足够快

目标：

# 10秒内完成记录

---

# 4️⃣ 摘要比数据更重要

用户真正需要的是：

# 「帮他整理病情」

而不是单纯数据堆叠。

---

# 七、病程摘要 UI 优化（重点）

# 当前问题

当前像：
- 数据列表
- 数据库记录

---

# 正确方向

应该像：

# 「系统帮用户整理好的病情变化」

---

# 推荐样式

```text
最近病程变化

✓ PSA 从 14 降至 12

✓ 已于 5月7日注射诺雷得

✓ 当前治疗状态稳定

下次复查：6月15日
```

---

# 设计重点

- 更像自然语言
- 更像总结
- 更少表格感

---

# 八、时间轴高级交互设计

# 核心原则

# 时间轴 = 产品核心

---

# 1️⃣ 时间线强化

## 当前问题

时间线存在感弱。

---

## 优化建议

### 时间线颜色

```css
#D8DCE3
```

---

## 节点设计

使用：
- 蓝色圆点
- 轻渐变
- 小阴影

增强扫描效率。

---

# 2️⃣ 事件颜色系统

不同类型不同颜色。

---

| 类型 | 颜色 |
|---|---|
| PSA | 蓝 |
| 打针 | 绿 |
| 检查 | 紫 |
| 用药 | 橙 |
| 复诊 | 青 |

---

# 3️⃣ 卡片信息密度优化

当前：
留白过大。

---

优化：
- 卡片更紧凑
- 信息分层
- 增强时间感

---

# 4️⃣ 快速记录交互

全局固定：

# 「＋记录按钮」

---

点击后弹出：

```text
选择记录类型：

- PSA记录
- 检查记录
- 打针记录
- 用药记录
- 复诊记录
- 备注记录
```

---

# 九、趋势分析页面优化

# 当前问题

页面太空。

---

# 优化方向

## 趋势图区域高度

占屏幕：

# 40%

---

## 图表建议

使用：
- 平滑折线
- 蓝色渐变区域
- 最大值/最小值标记

---

# 增加趋势总结（重要）

例如：

```text
近3次 TPSA 呈下降趋势
下降约 14%
```

---

# 风险提醒优化

避免：

```text
TPSA > 10
```

---

推荐：

```text
建议持续关注指标变化，
并结合医生意见定期复查。
```

---

# 十、iOS级别医疗产品风格方案

# 目标风格

参考方向：

- Apple Health
- Calm
- Gentler Streak
- 小睡眠
- Notion Mobile

---

# 核心风格关键词

- 极简
- 呼吸感
- 柔和
- 卡片化
- 克制
- 不压迫

---

# 设计重点

## 1️⃣ 大留白

让用户：
- 放松
- 不焦虑

---

## 2️⃣ 极轻阴影

减少：
- 后台系统感

---

## 3️⃣ 柔和动画

例如：
- 卡片渐入
- 数值轻变化
- 页面滑动过渡

---

## 4️⃣ 图标统一

推荐：
- 线性图标
- 圆角图标
- 柔和色块

---
